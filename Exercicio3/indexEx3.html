API's

O que é API de integração

Em termos simples, é uma interface que conecta duas ou mais aplicações, geralmente em JSON ou XML e garantem autenticação e segurança.

Beneficios:

  • Automação de processos

    Atividades repetitivas podem ser simplesmente automatizadas, como atualizar estoque no recebimento ou em uma venda.


  • Redução de erros manuais

    Em vez de depender de entradas manuais as APIs garantem informações constantes.


  • Mais eficiencia operacional

    Melhoria de fluxos de trabalho, permitindo que seja possivel equipes diferentes acessem uma ERP e CRM em tempo real


  • Comunicação facelitada entre sistemas diferentes

    Com ela, sistemas desenvolvidos em linguagens difetentes, sejam legados, sejam modernos


Funcionalidades:

  • EndPoints

    URLs específicas que representam recursos ou funcionalidades disponíveis para serem acessadas remotamente


  • Métodos HTTP

    Os métodos HTTP (GET, POST, PUT, DELETE, etc.) são utilizados para definir ações que podem ser realizadas nos endpoints


Tipos de APIs

  • Abertas

  • De parceiros

  • Internas

  • Compostas
